# AGENTS.md

Roles, responsibilities, and capabilities of the AI agent team.

## 1. ChatGPT (Architect & Orchestrator)
- **Role**: High-level design, system architecture, and continuity management.
- **Responsibility**: Maintaining `NOTES_CHATGPT.md`, defining long-term strategy, solving complex architectural blockers.
- **Strengths**: Broad context, reasoning, planning.

## 2. Gemini (Feature Generator & Code Writer)
- **Role**: Implementation specialist and multimodal expert.
- **Responsibility**: Writing code, integrating Gemini API, generating assets, documentation updates.
- **Strengths**: Large context window, code generation, speed.

## 3. Cursor (Local Executor & Integrator)
- **Role**: IDE-integrated development partner.
- **Responsibility**: Applying changes, running builds, debugging locally, indexing codebase.
- **Strengths**: Direct file access, diff application, context awareness.

## 4. Bolt (CI/Automation)
- **Role**: Automated validation and deployment.
- **Responsibility**: Running tests, checking builds, ensuring `metadata.json` is correct.
- **Strengths**: Precision, speed, repetition.

## 5. DeepSeek (Creative Strategist & Red-Team)
- **Role**: Critical analysis, creative direction, and risk assessment.
- **Responsibility**: Updating `REDTEAM.md`, brainstorming new features, finding logic gaps.
- **Strengths**: Deep reasoning, adversarial thinking.

## 6. Human (Project Owner)
- **Role**: Final decision maker and visual validator.
- **Responsibility**: Reviewing output, testing UI/UX, providing `USER_FEEDBACK.md`.
- **Communication**: Provides the "Session Starter" brief.

## 7. Communication Style
- **Concise**: Get to the point.
- **Structured**: Use Markdown headers and lists.
- **Explicit**: Reference specific files and lines of code.
